💬 Real-Time Chat Application

A modern real-time chat application built using HTML, CSS, and JavaScript. This project provides a clean UI and interactive features for seamless messaging between users.

🚀 Features
🔐 User login (enter username to join)
💬 Real-time messaging interface (frontend simulation)
👥 Chat rooms / chat list (dynamic UI support)
🟢 Online user status indicator
😊 Emoji picker with categories
🎤 Voice & attachment UI buttons (ready for extension)
🌙 Dark / Light theme toggle
📱 Fully responsive design
🛠️ Tech Stack
HTML5 – Structure of the application
CSS3 – Styling and layout (custom + responsive design)
JavaScript (Vanilla JS) – App logic and interactivity
Font Awesome – Icons
Google Fonts (Inter) – Typography
📂 Project Structure
REAL-TIME-CHAT-APPLICATION/
│── CHAT.html          # Main HTML file
│── stylechat.css      # Styling file

│── README.md          # Project documentation

⚙️ How It Works
User enters a name to join the chat.
Chat screen becomes active.
Messages appear in the chat container.
UI includes emoji picker, typing indicator, and chat controls.
Currently works as a frontend-based chat UI (no backend connected).
